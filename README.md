# eushaa-server
eushaa 프로젝트 api server

## 주요 기능

- 팀 매칭  : 간단한 성향 조사를 통해서 나와 유사한 팀원과 매칭!
- 투두리스트 작성 : 자유롭게 하루에 한번 완료!
- 캐릭터 꾸미기 : 자유롭게 표현하는 나만의 캐릭터!
- 우주 꾸미기 : 우주를 한 조각씩 맞춰가는 모두의 여정!
- 다함께 레벨 업! : 방대한 세계를 시끌벅적하게 탐험하라!

## 기술스택

- 백엔드
    - node.js
    - express
    - graphql-yoga
    - mysql
    - sequalize
